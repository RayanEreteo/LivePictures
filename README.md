# Live Pictures - Client

Client code for "Live Pictures"

[BACKEND CODE](https://github.com/RayanEreteo/LivePictures_Server)

## Overview

Live Pictures is a real-time image sharing application. This repository contains the client-side implementation.

## Features

- Real-time image upload
- No account required
- Live preview of shared content
- Simple User-friendly interface
- Cross-platform compatibility

## Installation

```bash
git clone https://github.com/RayanEreteo/LivePictures
cd LivePictures
npm install
```

## Usage

Set backend URL on **.env** then run :

```bash
npm run dev
```

## Technologies

- Javascript
- WebSocket
- Socket.io
